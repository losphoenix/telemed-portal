# App Routing Notes

Recommended route groups or navigator stacks:

- `auth`
- `onboarding`
- `home`
- `chat`
- `appointments`
- `messages`
- `follow-up`
- `billing`
- `profile`

Suggested screen ownership:

- `home`: care dashboard, upcoming visit, quick actions
- `chat`: AI concierge, structured triage, routed booking handoff
- `appointments`: list, booking flow, detail, telehealth join
- `messages`: thread list and conversation detail
- `follow-up`: scheduled check-ins and escalation states
- `billing`: co-pay and receipts
- `profile`: demographic, insurance, settings
